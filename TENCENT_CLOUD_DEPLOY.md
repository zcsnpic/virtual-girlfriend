# 腾讯云部署指南

本指南将帮助你在腾讯云平台部署虚拟女友Web应用。

## 方法一：对象存储 COS + CDN（推荐）

### 步骤1：登录腾讯云控制台
1. 访问 [腾讯云官网](https://cloud.tencent.com/)
2. 登录你的腾讯云账号

### 步骤2：创建存储桶
1. 进入 **对象存储 COS** 控制台
2. 点击 **创建存储桶**
3. 填写配置信息：
   - 存储桶名称：自定义（如 `virtual-girlfriend`）
   - 所属地域：选择靠近你的地域
   - 访问权限：设置为 **公有读私有写**
   - 其他默认即可
4. 点击 **确定** 完成创建

### 步骤3：开启静态网站功能
1. 在存储桶列表中找到刚创建的存储桶，点击进入
2. 选择 **基础配置** > **静态网站**
3. 点击 **编辑**，开启静态网站功能
4. 设置：
   - 索引文档：`index.html`
   - 错误文档：`index.html`（单页应用需要）
5. 点击 **保存**

### 步骤4：上传文件
1. 选择 **文件列表** > **上传文件**
2. 上传以下文件和文件夹：
   - `index.html`
   - `css/` 文件夹（包含所有CSS文件）
   - `js/` 文件夹（包含所有JS文件）
3. 确保文件结构正确

### 步骤5：获取访问域名
1. 在 **静态网站** 配置页面，复制 **访问域名**
2. 通过该域名即可访问你的应用

## 方法二：轻量应用服务器

### 步骤1：创建轻量应用服务器
1. 进入 **轻量应用服务器** 控制台
2. 点击 **创建实例**
3. 选择配置：
   - 地域：选择靠近你的地域
   - 镜像：选择 **静态网站** 镜像
   - 套餐：根据需求选择合适的套餐
4. 点击 **购买** 完成创建

### 步骤2：登录服务器
1. 在实例列表中找到刚创建的实例，点击 **登录**
2. 使用提供的登录方式进入服务器

### 步骤3：上传文件
1. 使用 FTP 工具（如 FileZilla）连接服务器
2. 将应用文件上传到网站根目录（通常是 `/www/wwwroot/`）

### 步骤4：访问应用
1. 在实例列表中复制服务器的 **公网IP**
2. 在浏览器中输入该IP地址访问应用

## 方法三：云开发静态网站托管

### 步骤1：创建云开发环境
1. 进入 **云开发 CloudBase** 控制台
2. 点击 **创建环境**
3. 填写配置信息：
   - 环境名称：自定义（如 `virtual-girlfriend`）
   - 环境类型：选择 **标准环境**
   - 其他默认即可
4. 点击 **确定** 完成创建

### 步骤2：开启静态网站托管
1. 在环境列表中找到刚创建的环境，点击进入
2. 选择 **静态网站** > **开始使用**
3. 点击 **上传文件**，上传应用文件

### 步骤3：获取访问域名
1. 上传完成后，复制 **默认域名**
2. 通过该域名即可访问你的应用

## 注意事项

1. **HTTPS 配置**：腾讯云提供免费的 SSL 证书，建议开启 HTTPS 以提升安全性

2. **域名绑定**：如果你有自己的域名，可以在腾讯云控制台进行域名绑定

3. **CDN 加速**：对于静态网站，建议开启 CDN 加速以提升访问速度

4. **定期备份**：定期备份你的应用文件，以防数据丢失

5. **监控告警**：开启必要的监控告警，及时发现并解决问题

## 常见问题

### 1. 上传文件后访问404
- 检查文件路径是否正确
- 确认静态网站功能已开启
- 检查索引文档设置是否正确

### 2. 页面加载缓慢
- 开启 CDN 加速
- 优化图片资源
- 考虑使用腾讯云的全球加速服务

### 3. 无法访问应用
- 检查存储桶权限设置
- 确认网络连接正常
- 查看腾讯云控制台的错误日志

## 部署完成

部署完成后，你可以通过腾讯云提供的域名访问虚拟女友应用。如果需要更新应用，只需重新上传修改后的文件即可。
