## 自动发送逻辑分析

### 核心设计：智能自动发送系统

**设计理念**：
- **手打输入**：等待用户停止输入后自动发送（延迟发送）
- **粘贴/语音**：立即发送，不等待延迟
- **中文输入法**：通过 composition 事件检测组合输入，避免误发

### 关键技术点

#### 1. 中文输入法检测
```javascript
// compositionstart - 开始组合输入（如拼音输入法）
messageInput.addEventListener('compositionstart', () => {
    this.isComposing = true;
    this.clearAutoSendTimer();
});

// compositionend - 结束组合输入
messageInput.addEventListener('compositionend', (e) => {
    this.isComposing = false;
    this.handleSmartAutoSend(e);
});
```

**作用**：防止在输入拼音等组合输入时触发自动发送

#### 2. 空格检测机制
```javascript
const lastChar = value[value.length - 1];
const isLastCharSpace = lastChar === ' ';

if (isLastCharSpace) {
    // 最后一个字符是空格，不自动发送，等待用户继续输入
    this.clearAutoSendTimer();
}
```

**作用**：用户输入空格表示还想继续输入，暂缓自动发送

#### 3. 自动发送延迟
```javascript
const autoSendDelay = parseFloat(settings.autoSendDelay || 2.5); // 默认 2.5 秒
const delayMs = autoSendDelay * 1000;

this.autoSendTimer = setTimeout(() => {
    // 延迟后自动发送
    this.sendMessage();
}, delayMs);
```

**用户设置**：0.5-3 秒可调

#### 4. 计时器管理
```javascript
clearAutoSendTimer: function() {
    if (this.autoSendTimer) {
        clearTimeout(this.autoSendTimer);
        this.autoSendTimer = null;
    }
}
```

**触发时机**：
- 每次输入都重置计时器
- 最后一个字符是空格时清除
- 用户手动发送时清除

### 工作流程

```
用户输入 → isComposing检查 → 最后字符检查 → 重置计时器 → 延迟等待 → 自动发送
                                              ↓
                                          输入/空格 → 清除计时器 → 重新开始
```

### 特殊处理

1. **空输入**：清除计时器，不发送
2. **组合输入中**：跳过自动发送逻辑
3. **最后字符为空格**：取消自动发送，等待继续输入
4. **手动发送/Enter**：清除计时器，立即发送
5. **粘贴操作**：立即发送（通过 input 事件检测）

### 优点

✅ **中文友好**：完美支持中文输入法组合
✅ **智能判断**：通过空格检测用户意图
✅ **可配置延迟**：0.5-3 秒自由调整
✅ **防误发**：多重检查机制
✅ **手动优先**：随时可手动发送打断
✅ **输入体验好**：不需要频繁点击发送按钮

### 配置项

- **autoSendDelay**：自动发送延迟（默认 2.5 秒）
- 设置界面：`设置 → 对话设置 → 自动发送延迟`
- 范围：0.5-3.0 秒

## 自动发送功能开关添加

### 修改内容

**时间**: 2026-03-24 09:55

**修改文件**:

2. `js/app.js`

### 具体修改

#### 1. HTML 修改 (index.html)
- 在自动发送延迟设置上方添加"启用自动发送"开关
- 自动发送延迟设置添加 ID `autoSendDelayGroup`，用于控制显示/隐藏
- 默认状态：启用（checked）

#### 2. JavaScript 修改 (js/app.js)

**loadSettings() 函数**:
- 加载 `autoSendEnabled` 开关状态
- 根据开关状态控制 `autoSendDelayGroup` 的显示/隐藏

**bindEvents() 函数**:
- 添加 `autoSendEnabled` 开关的 `change` 事件监听
- 当开关切换时，动态显示/隐藏延迟设置

**handleSmartAutoSend() 函数**:
- 在函数开头添加开关状态检查
- 如果 `autoSendEnabled === false`，直接返回，跳过所有自动发送逻辑

**saveSettings() 函数**:
- 保存 `autoSendEnabled` 状态到设置中

### 功能特性

✅ **默认启用**：保持原有行为，向后兼容
✅ **即时生效**：切换开关后立即生效
✅ **UI 联动**：关闭开关时，延迟设置自动隐藏
✅ **持久化**：开关状态保存到 localStorage
✅ **无侵入**：不影响其他功能

### 使用方法

1. 打开设置（点击 ⚙️ 按钮）
2. 找到"启用自动发送"开关
3. 切换开关以开启/关闭自动发送功能
4. 关闭时，自动发送延迟设置会自动隐藏
